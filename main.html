Política de Privacidade

Última atualização: 10 de fevereiro de 2025

Esta Política de Privacidade descreve como a WhiteShop coleta, usa e protege as informações dos usuários que utilizam o aplicativo de automação de vídeos integrado ao TikTok via n8n.

1. Informações Coletadas

Ao conectar sua conta TikTok, a WhiteShop recebe dados fornecidos pela API oficial, incluindo:

  • Tokens de autenticação (access token e refresh token);
  • Identificador do usuário (open_id).

Essas informações são necessárias apenas para executar automações e postar vídeos conforme solicitação do usuário.

2. Uso das Informações

Os dados coletados são utilizados exclusivamente para:

  • Autenticar sua conexão com a API do TikTok;
  • Realizar uploads e postagens automáticas de vídeos;
  • Executar os fluxos configurados dentro do n8n.

3. Retenção e Exclusão de Dados

Os tokens de acesso são armazenados de forma temporária e segura apenas enquanto necessários para a execução das automações. O usuário pode solicitar a exclusão de suas informações a qualquer momento pelo e-mail contato@whiteshop.com ou revogar o acesso diretamente em sua conta TikTok.

4. Compartilhamento de Dados

A WhiteShop não compartilha nem comercializa informações de usuários com terceiros, exceto quando exigido por lei.

5. Segurança

Adotamos medidas técnicas e administrativas adequadas, como conexões seguras (HTTPS) e armazenamento protegido, para evitar acesso não autorizado ou vazamento de dados.

6. Alterações a Esta Política

Esta Política poderá ser atualizada periodicamente. As alterações serão publicadas nesta página com nova data de atualização.

7. Contato

Para dúvidas, solicitações ou exclusão de dados, entre em contato: contato@whiteshop.com
